# p4docker
To deploy [Perforce](https://www.perforce.com/) VCS is a piece of cake, with a one-liner command, you are ready to go.



### How To Use

```shell
docker compose up -d
```



### Credits

Mainly comes from [this blog post](https://oualid.games/blogs/dockerize-your-perforce/), and I made some minor changes to adapt to the latest Perforce versions.
